## **Application Overview**

This is an AI-powered review platform designed to evaluate newly onboarded employees based on video learning materials. The system simulates an interactive interview environment where users respond to topic-based questions, and their performance is analyzed in real time.

The web application consists of two main interfaces:

* **User Interface**
* **Admin Interface**

---

## **User Features**

* **Authentication**

  * Secure login functionality
  * Forgot password and password reset options

* **Topic Selection**

  * Users can select review topics from a predefined list
  * Each session requires a minimum of 5 and a maximum of 15 topics

* **AI-Powered Review Session**

  * Users are guided to a review (interview) page
  * Real-time interaction with an AI reviewer
  * Live transcription of both user responses and AI prompts

* **Session Management**

  * Users can end the session at any time

* **Progress Tracking**

  * A dedicated progress page displays:

    * Performance reports from previous sessions
    * Overall progress and improvement trends

---

## **Admin Features**

* **Analytics Dashboard**

  * Key metrics and insights, including:

    * Total users
    * Users who completed all reviews
    * Users currently in progress
    * Total sessions conducted
    * Passed vs. failed sessions
    * User profile distribution (visualized via pie charts)
    * Session activity over the last 30 days
    * VAPI call cost analytics
    * Evaluation cost analytics
    * Total operational cost

* **User Management**

  * View user list with pagination
  * Add new users (name, email, profile)
  * Delete existing users

* **User Details & Reports**

  * Access individual user profiles, including:

    * Name, email, and profile information
    * Detailed reports of all previous sessions
